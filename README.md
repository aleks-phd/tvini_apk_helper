# APK Installer on Android Devices
* This is application to help easily install APK files on Android device
* This source code has implementation of custom logic to help with installin APKs on RayNeo X3 AR glasses
* Please follow the instructions below to compile and launch the code
  - Make sure you have Python3.13
  - create folder: toos/windows/
  - Downaod scrcpy for Windows: https://github.com/Genymobile/scrcpy
  - Extract archive and put it under toos/windows/, make sure folder name is: scrcpy
  - download Zadig tool: https://zadig.akeo.ie/downloads/
  - Put exe file under  toos/windows/ folder
  - run python code, all logic is in a single file
